## üöÄ Quick Start

### Requirements

- Node.js 18+ 
- pnpm (recommended) / npm / yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Moyuin-aka/tyndall-public
cd tyndall-public

# Install dependencies
pnpm install

# Start dev server
pnpm dev  # Visit http://localhost:4321

# Production build
pnpm build

# Preview build
pnpm preview
```
